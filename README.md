# DeepThinkingFlow-AI

Focused on structured reasoning, bilingual (vi/en) behavior, and adapter-based fine-tuning via LoRA/QLoRA. Includes a unified CLI, multi-turn chat terminal, and a full training pipeline — 7/7 smoke tests passing.
